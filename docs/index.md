# Google Ad Manager Provider

Manage Google Ad Manager (GAM) configuration — ad units, placements, and custom targeting — declaratively through the [Ad Manager REST API](https://developers.google.com/ad-manager/api/beta).

~> **Destroy means archive.** Google Ad Manager does not hard-delete most entities. `terraform destroy` archives ad units and placements and deactivates custom targeting keys. Set `skip_archive_on_destroy = true` on a resource to remove it from state without touching GAM.

## Authentication

The provider authenticates with a Google service account, either via a JSON key (the `credentials` attribute) or [Application Default Credentials](https://cloud.google.com/docs/authentication/application-default-credentials). The service account must be added as a user of the Ad Manager network, and API access must be enabled for the network (Ad Manager UI → Admin → Global settings).

## Example Usage

```terraform
provider "admanager" {
  network_code = "123456"           # or set ADMANAGER_NETWORK_CODE
  credentials  = "/path/to/sa.json" # or use Application Default Credentials

  # Optional: Ad Manager API quotas are low; defaults are conservative.
  requests_per_second = 2
  retry_max_attempts  = 5
}
```

## Schema

### Optional

- `credentials` (String, Sensitive) Path to (or JSON content of) a Google service account key. The service account must be added as a user of the Ad Manager network. When unset, [Application Default Credentials](https://cloud.google.com/docs/authentication/application-default-credentials) are used (which honor `GOOGLE_APPLICATION_CREDENTIALS`).
- `network_code` (String) The Ad Manager network code to manage (e.g. `123456`). Can also be set with the `ADMANAGER_NETWORK_CODE` environment variable.
- `requests_per_second` (Number) Client-side rate limit for Ad Manager API calls, shared by all resources. Ad Manager quotas are low; the default of `2` is conservative on purpose. Raise it only if your network's quota allows.
- `retry_max_attempts` (Number) Maximum attempts per API call (initial call + retries) for rate-limit and transient errors. Defaults to `5`.
